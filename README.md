# dotfiles

## Install

### Ubuntu

```bash
sudo apt update
sudo apt install git

git clone https://github.com/HamletSargsyan/dotfiles.git

cd dotfiles
./builder.sh
```

### Termux

```bash
pkg update
pkg install git

git clone https://github.com/HamletSargsyan/dotfiles.git

cd dotfiles
./builder.sh
```
